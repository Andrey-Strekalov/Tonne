# Fermer-front

Фронтенд приложения Fermer (React + Vite + TypeScript).

**Стек:** React 19, Vite 8, TypeScript, React Router 7, TanStack Query, Tailwind CSS 4, Radix UI.

## Как запустить проект

### 1. Клонировать репозиторий

```bash
git clone <URL-репозитория>
cd Fermer-front
```

### 2. Установить зависимости

```bash
npm install
```

### 3. Запустить проект

```bash
npm run dev
```

Приложение откроется по адресу (обычно `http://localhost:5173`).

---

## Сборка

```bash
npm run build
```

## Скрипты

| Команда   | Описание        |
|----------|-----------------|
| `npm run dev`     | Режим разработки |
| `npm run build`   | Продакшн-сборка   |
| `npm run preview` | Просмотр сборки   |
